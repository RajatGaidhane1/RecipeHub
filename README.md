# MyRecipeApp

MyRecipeApp is an Android application built in Kotlin using Jetpack Compose and MVVM architecture. It demonstrates consuming RESTful APIs using Retrofit and Coroutines by fetching and displaying meals from the [Meal DB API](https://www.themealdb.com/). Developed as a **learning project through a structured Android development course**, it showcases the ability to implement modern Android development patterns and clean architecture.

## Features

- Displays a list of meals fetched from a RESTful API.
- Shows meal images alongside their names in a clean, scrollable list.
- Read-only application: users can browse meals but cannot modify data.
- Smooth, responsive UI built entirely with Jetpack Compose.
- Uses Coroutines for efficient asynchronous API calls.

## Tech Stack

- **Language:** Kotlin  
- **Architecture:** MVVM (Model-View-ViewModel)  
- **UI:** Jetpack Compose  
- **Networking:** Retrofit + RESTful APIs  
- **Asynchronous handling:** Coroutines  
- **Navigation:** Jetpack Navigation Component
